# ✨App Description
> 사용자가 업무와 일정을 효율적으로 관리할 수 있도록 돕는 개인화된 일정 관리 앱 <br><br>
> 중요도 설정 및 정렬, 추천 순위 기능 등을 통해 생산성과 관리 효율성을 높이는 데 중점을 둠 <br><br>
> 회원가입 및 로그인 기능과 함께, 일정 작성, 삭제, 수정, 검색, 정렬 등 다양한 기능 제공 <br><br>
